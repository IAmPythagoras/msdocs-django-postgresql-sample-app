FFXIV Combat Simulator

THIS WEBSITE IS STILL IN DEV. IF YOU SEE A BUG PLEASE LET ME KNOW THROUGH DISCORD

FFXIV Combat Simulator (aka ffxivcalc) is a python library that allows one to simulate combat from the game Final Fantasy XIV Online. It lets you simulate any number of players of any jobs (except Blue Mage) and will tell you every player's expected DPS (and more). The simulator is also able to let you know if the actions you are taking are illegals, in which case the simulation will stop and notify you (note that you can disable that checking). Start simulating you wildest fantasy and see the outcome for yourself! And if you are having any questions join the discord by clicking on the picture above to get help.

Note that the simulator is still in active development and might be the subject of some bugs. It is recommended to interpret all that the simulator says with a grain of salt, but it is still a very good way to see if doing something is better and also gives a close indication of how in game damage is going to look like. If you are interested in how the simulator works behind the scene and also on the other things it can do, I recommend you check out the GitHub page by clicking the GitHub picture above. There are a lot of customizable things that the online version of the simulator does not allow, but by using the python library directly you can do a lot more.
