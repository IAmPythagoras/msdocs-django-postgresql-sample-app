FFXIV Combat Simulator

THIS WEBSITE IS STILL IN DEV. IF YOU SEE A BUG PLEASE LET ME KNOW THROUGH DISCORD

FFXIV Combat Simulator (aka ffxivcalc) is a python library that allows one to simulate combat from the game Final Fantasy XIV Online. It lets the user simulate any number of players of any job (except Blue Mage) and shows every player's expected DPS as well as other combat statistics. The simulator is also able to detect if a chosen action is illegal, in which case the simulation will stop and notify the user; you can disable this option in the simulation page. If you have any questions, feel free to join the Discord server dedicated to the project by clicking the Discord logo above.

Note that the simulator is still in active development and bugs may be encountered. The results may not perfectly replicate in-game performance (for instance, the program does not account for lag and assumes an average crit rate),but they will give a very good idea of how in game damage would look like given a set of actions.

If you are interested in how the simulator works behind the scenes and other things it can do, I recommend checking out the GitHub page by clicking the GitHub logo above. By using the python library directly, you can customize several settings, which this website does not allow.
